# AURA Static Site

Deploy the contents of this `site` folder as a static website.

Required files:
- `index.html`
- `philosophy.html`
- `security.html`
- `styles.css`
- `app.js`
- `assets/`

The site uses relative paths, so it works on static hosts such as Netlify, Vercel, GitHub Pages, or any normal web server. Use `index.html` as the home page.
